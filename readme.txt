1. Have installed Python version 3.9 and later
2. In project folder, create own virtual environment (preferably)
3. To virtual environment, install requirements using PIP. To do so:
4. Type command: pip install -r requirements.txt
5. Download webdriver file and have it placed in arbitrary separate directory
6. Before downloading, match the driver version with browser version
7. URL to webdriver: https://chromedriver.chromium.org/downloads
8. Include webdriver containing folder in path variable, in system enviromental variables
9. To run the test, change directory to ./Tests
10. Type command: robot heureka.robot
